Polisi masih menunggu hasil Pusat Laboratorium Forensik (Puslabfor) Bareskrim Polri untuk mengetahui penyebab kebakaran besar di Jalan Kemayoran Gempol, Jakarta Pusat. Kapolres Jakarta Pusat, Kombes Susatyo Purnomo, menampik kabar bahwa ada oknum yang sengaja membakar hunian di kawasan itu. Sebelumnya, saksi mengaku api bersumber dari plafon rumah salah satu warga dan mencium bau kabel terbakar sebelum asap menebal. Empat orang saksi sudah dimintai keterangan. Sementara itu, Kasudin Damkar Jakpus Asril mengungkapkan dugaan api berasal dari ledakan kompor salah satu rumah warga. Kebakaran menghanguskan 543 bangunan rumah tanpa korban jiwa.
