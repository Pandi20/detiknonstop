**Arti Kode Kursi Ekonomi di Kereta Api**

Kode subkelas ekonomi di kereta api jarak jauh, seperti C, CA, CB, CC, P, Q, dan S, memiliki tarif berbeda berdasarkan waktu pemesanan. KAI membagi subkelas ini untuk memberikan opsi harga terjangkau bagi penumpang. Pemesanan awal lebih menguntungkan karena harga lebih murah. Fasilitas, layanan, dan jenis kursi di setiap subkelas ekonomi sama, hanya berbeda dalam harga tiket.
