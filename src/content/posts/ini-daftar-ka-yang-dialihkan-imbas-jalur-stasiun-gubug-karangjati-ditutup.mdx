PT Kereta Api Indonesia (KAI) menutup jalur kereta api antara Stasiun Gubug dan Stasiun Karangjati di Kabupaten Grobogan akibat luapan air. Langkah ini diambil untuk menjaga keselamatan perjalanan. Beberapa kereta api dialihkan melalui jalur alternatif. Penutupan jalur dilakukan pada Jumat, 24 Januari 2025, pukul 22.25 WIB, karena luapan air menggenangi area jalur. KAI mengerahkan tim tanggap darurat untuk penanganan. Daftar kereta api terdampak termasuk KA Kertajaya, Dharmawangsa, Pandalungan, Sembrani, Argo Bromo Anggrek, Blambangan Ekspres, Jayabaya, Gumarang, dan Harina. KAI meminta maaf atas ketidaknyamanan dan berjanji untuk segera menyelesaikan penanganan demi keselamatan dan kenyamanan perjalanan.
