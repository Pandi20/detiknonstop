PT Kereta Api Indonesia (KAI) Daerah Operasi 1 (Daop 1) Jakarta mencatat peningkatan penumpang menjelang libur Hari Isra Mi'raj dan Imlek. Hingga pukul 07.00 WIB, 68 perjalanan kereta api jarak jauh dengan kapasitas 37.024 tempat duduk telah beroperasi. Tingkat okupansi mencapai 100,5% dengan total tiket terjual 37.198 tempat duduk. Sejak Jumat (24/1), 37.383 penumpang telah diberangkatkan.

Namun, perjalanan terganggu akibat luapan air di jalur Gubug-Karangjati, Kabupaten Grobogan, pada Jumat (24/1) pukul 22.25 WIB. Jalur tersebut ditutup sementara untuk keselamatan. KAI telah menyiagakan petugas dan alat berat untuk menangani gangguan ini. Beberapa kereta api dialihkan melalui jalur alternatif, seperti Brumbung-Gundih-Gambringan dan Brumbung-Solo-Surabaya.
